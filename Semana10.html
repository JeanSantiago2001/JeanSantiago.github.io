¿Qué Temas Aprendió?

DESARROLLO DE APLICACIONES WEB

En el desarrollo de React, se adquirio conocimientos clave al utilizar el hook useState, permitiéndome incorporar variables de estado en mis componentes para lograr la persistencia del estado entre renderizaciones y la actualización dinámica de la interfaz de usuario. Además, exploré el uso de JSX, una extensión de JavaScript empleada en React para organizar la interfaz mediante componentes reutilizables. Entendí el proceso de renderizado en React, utilizando ReactDOM.render() para mostrar elementos HTML en la aplicación. Asimismo, aprendí a gestionar la transferencia de datos entre componentes, ya sea de manera unidireccional (de padre a hijo o viceversa) o entre componentes anidados. La recomendación de estructurar los componentes en archivos separados fue entendida para mejorar la reutilización y el mantenimiento del código. Finalmente, utilicé la biblioteca Axios para realizar solicitudes HTTP y consumir datos de la API de Rick and Morty, demostrando así la integración efectiva de APIs externas en mis proyectos de React..

UseState

Un hook de React que permite agregar variables de estado a componentes funcionales, facilitando la gestión de datos que pueden cambiar a lo largo del ciclo de vida del componente.

JSX

Una extensión de JavaScript utilizada en React para escribir de manera más concisa y legible la estructura de la interfaz de usuario, combinando HTML y JavaScript.

Axios

Una biblioteca de JavaScript utilizada para realizar solicitudes HTTP, comúnmente empleada en el consumo de APIs.

API

Es una interfaz de procesamiento de aplicaciones entre un servidor web y un navegador web, para el ejemplo se uso una API en especifico, de la serie de televisión "Rick and Morty".

PROCEDIMIENTOS

DESARROLLO DE APLICACIONES WEB

Implementación de UseState: Se creo un componente utilizando el hook useState para gestionar y mostrar un contador que se incrementa al hacer clic en un botón.

Formulario con UseState: Se desarrollo un formulario que utiliza useState para recolectar y mostrar información ingresada por el usuario, demostrando cómo manejar datos en tiempo real.

Consumo de API con Axios: Se utilizo la biblioteca Axios para realizar solicitudes a la API de Rick and Morty, mostrando dinámicamente información de personajes en función de la respuesta de la API.
