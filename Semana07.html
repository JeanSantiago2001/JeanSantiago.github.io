Como funciona la web, Que aplicaciones existen

La internet es un sistema de información a nivel mundial que posibilita el acceso y compartición de contenido a través de la red. Este funcionamiento se basa en la interacción entre servidores y clientes: los servidores almacenan y manejan la información, mientras que los clientes, como los navegadores web, solicitan y presentan dicha información a los usuarios.

Entre las aplicaciones más frecuentes de internet se encuentran:

1. Correo electrónico

2. Redes sociales

3. Motores de búsqueda

4. Comercio electrónico

5. Streaming de contenido.

Tecnologías web

Las tecnologías web se refieren a las herramientas y lenguajes utilizados en la creación de aplicaciones y sitios web. Estas tecnologías se dividen principalmente en dos categorías: frontend y backend. El frontend se enfoca en la parte visible de un sitio web, mientras que el backend se encarga de la lógica y el procesamiento de datos en el servidor.

Algunas de las tecnologías web más populares son:

Frontend: HTML, CSS, JavaScript

Backend: PHP, Ruby, Java, Python

Además de estos lenguajes, existen frameworks que agilizan el desarrollo web y proporcionan soluciones para desafíos comunes. Algunos ejemplos de frameworks populares incluyen:

Para Backend: Symfony, Laravel, Phalcon, Django, Ruby on Rails, .NET Framework

Diseño de experiencia de interfaz

El Diseño de Experiencia de Interfaz (UX/UI) se concentra en la creación de productos digitales que sean fáciles de utilizar, intuitivos y agradables para los usuarios. Este enfoque se orienta hacia la experiencia completa del usuario, considerando sus percepciones, emociones y reacciones ante el producto, sistema o servicio de una empresa. Se compone de dos aspectos principales:

1. Diseño de Experiencia de Usuario (UX): Este aspecto se enfoca en el recorrido que sigue el usuario dentro de una aplicación o proceso, y en la forma y función general de un producto o tecnología. El objetivo del UX es asegurar que la navegación y las acciones que realiza el usuario sean coherentes y fáciles de comprender.

2. Diseño de Interfaz de Usuario (UI): El diseño UI se concentra en la apariencia visual y la funcionalidad externa de un producto. El diseñador UI decide cómo se verá la interfaz y cómo funcionará visualmente. A diferencia del UX, que se enfoca en los procesos y acciones del usuario, el UI se centra en el diseño visual y la interacción tangible del usuario con la interfaz.

PROCEDIMIENTOS

DESARROLLO DE APLICACIONES WEB

1. Identificar el número IP de nuestra PC:

Para obtener la dirección IP de tu equipo en Windows, puedes abrir la línea de comandos (CMD) y escribir el comando ipconfig. Este comando te mostrará información detallada sobre la configuración de red de tu PC, incluyendo la dirección IP asignada a tu dispositivo.

2. Comprobar la conectividad del servidor:

Para verificar la conectividad con un servidor específico, puedes usar el comando ping seguido de la dirección IP del servidor o su nombre de dominio. Por ejemplo, escribe ping 192.168.1.1 o ping www.ejemplo.com y presiona Enter. Esto enviará paquetes de datos al servidor y recibirás respuestas para confirmar la conexión.

3. Verificar el recorrido de servidores:

Para rastrear la ruta que sigue tu conexión hacia un sitio web o servidor en particular, puedes usar el comando tracert seguido de la dirección web o IP del servidor. Por ejemplo, escribe tracert www.ejemplo.com o tracert 192.168.1.1 y presiona Enter. Este comando mostrará una lista de todos los servidores o nodos intermedios que tu conexión atraviesa antes de llegar al destino final, proporcionando información sobre la ruta de conexión y los tiempos de respuesta de cada nodo.
